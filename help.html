Application Help

This application is used for comparing invoices and purchase orders. Here's how it works:

  1. Users upload both the invoice and purchase order documents in PDF format.
  2. The application extracts text data from the uploaded documents using Google Document AI.
  3. The extracted text data is then sent to the Gemini LLM Module for comparison.
  4. The LLM module compares the data from the invoice and purchase order based on items and total amounts.
  5. If there are no discrepancies, the user can process the payment directly.
  6. If discrepancies are found, the user can send the documents for manual review.
  7. During manual review, the user can add comments and decide whether to proceed with the payment or not.

If you encounter any issues or need further assistance, please contact the support team.
